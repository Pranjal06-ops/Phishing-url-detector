# Contributing to Phishing URL Detector

Contributing

We’re excited to have you contribute! Whether it’s fixing a bug, adding a feature, or improving documentation, your help is appreciated. To make collaboration smooth, please follow these steps:

Fork the repository and create a feature branch:

git checkout -b feature/your-feature-name


Write clear commit messages that explain what you’ve changed.

Add tests for any new features or fixes in the tests/ folder.

Follow Python best practices (PEP8) to keep the code clean and consistent.

Open a Pull Request describing your changes, why they’re needed, and any related issues.

Every contribution counts, no matter how small! Thank you for helping improve this project.

Thank you for contributing!
